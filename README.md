# P.DNVD® — Independent Creative Studio

Portfolio/agencia creativa construido con React, Three.js, GSAP y Vite.

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
```

La carpeta `dist/` contiene la versión lista para desplegar en un hosting estático.

## Antes de publicar

- Sustituir `hello@pdnvd.com` por el correo definitivo.
- Enlazar los casos de estudio y perfiles sociales reales.
- Añadir fotografías o renders finales dentro de cada universo si se quiere ampliar la narrativa.
